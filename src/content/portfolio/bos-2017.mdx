La especialización en maquetaciones web avanzadas siguiendo estándares nos proporcionó la colaboración
para realizar la integración de un servicio externo en diferentes portales de entidades financieras.

Nuestro trabajo consistió en maquetar cabeceras, pieceras y la propia aplicación para ser embebida en los 
portales de las entidades. Para ello, se realizó maquetación pixel perfect, responsive y con distintos temas 
CSS para estandarizar la aplicación a embeber y que la navegación en los portales fuese exacta a la web 
donde se alojase.

La maquetación del contenido era común a todos los portales salvo por los colores propios de la entidad 
donde se incluyese la sección de fondos cambiando los colores primario, secundario y acento para los botones, 
separadores, etc.

Para la realización de estas maquetaciones web se utilizó Bootstrap con un compilado propio configurando cada una 
de las variables con los colores, espaciados, tipos de letra y demás requerimientos por parte de las entidades.
De esta forma, si era necesario ampliar los componentes o elementos del portal ya existía un marco de trabajo
realizado con un "estándar".
